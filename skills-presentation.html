סקיל #1
🎨 ממשקי משתמש מרהיבים
הסקיל שהופך Claude למעצב UX מקצועי
מה זה?
יוצר רכיבי UI ייחודיים ומקצועיים, נמנע מ"עיצוב AI גנרי". עובד עם HTML, React, Vue — ולכל הקשר סגנון משלו.
למה זה שווה?
טיפוגרפיה, אנימציות, פלטות צבעים — כמו מעצב UX אמיתי, בשבריר הזמן. בוחר סגנון עיצובי ומבצע אותו עם דיוק.
דוגמה חיה 🚀
המצגת הזו נוצרה עם הסקיל הזה — ממצגת Space Invaders רטרו ועד עיצוב מקצועי נאמן לקווי מנורה.

סקיל #2
🔌 בניית אפליקציות AI
Anthropic SDK · Prompt Caching · כל המודלים
מה זה?
בונה ומדבג אפליקציות עם Claude API, כולל Tool Use, Managed Agents ו-streaming. מכיר את ה-SDK לעומק.
Prompt Caching אוטומטי
כל אפליקציה שנבנית מקבלת caching מובנה. חיסכון של עד 90% בעלות על הודעות חוזרות וקונטקסט ארוך.
מעבר בין מודלים
Opus 4.7 · Sonnet 4.6 · Haiku 4.5 — הסקיל מנהל migration בין גרסאות ללא כאב, כולל תמיכה במודלים שיצאו משימוש.
